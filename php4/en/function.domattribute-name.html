DomAttribute::name

(PHP 4 >= 4.1.0)

DomAttribute::name — Returns the name of attribute

Description

string DomAttribute::name ( void )

Gets the name of the attribute.

Return Values

Returns the name of the attribute.

Migrating to PHP 5

Use the name property of DOMAttr.
