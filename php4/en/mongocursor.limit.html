MongoCursor::limit — Limits the number of results returned

Description

public MongoCursor MongoCursor::limit ( int $num )

Parameters

num

The number of results to return.

Return Values

Returns this cursor.

Errors/Exceptions

Throws MongoCursorException if this cursor has started iterating.
