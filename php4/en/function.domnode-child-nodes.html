DomNode::child_nodes

(PHP 4 >= 4.1.0)

DomNode::child_nodes — Returns children of node

Description

array DomNode::child_nodes ( void )

Returns all children of the node.

See also domnode_next_sibling(), and domnode_previous_sibling().
