http_send_data

(PECL pecl_http >= 0.1.0)

http_send_data — Send arbitrary data

Description

bool http_send_data ( string $data )

Sends raw data with support for (multiple) range requests.

Parameters

data

data to send

Return Values

Returns TRUE on success or FALSE on failure.
