HttpResponse::getCache

(PECL pecl_http >= 0.10.0)

HttpResponse::getCache — Get cache

Description

static bool HttpResponse::getCache ( void )

Get current caching setting.

Return Values

Returns TRUE if caching should be attempted, else FALSE.
